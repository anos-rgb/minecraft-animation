Getting Started

1. Upload a Minecraft skin or use the default skin

2. Create a new animation using the Animation Settings

3. Add keyframes to the timeline

4. Adjust bone positions, rotations, and scales

5. Preview your animation and export as JSON

Timeline Controls

- Drag the timeline marker to scrub through the animation

- Add keyframes to save bone positions at specific times

- Use playback controls to preview your animation

Bone Controls

- Use sliders to adjust bone rotations, positions, and scales

- Switch between tabs for different transform types

- Apply presets for common animations

Export Options

- Copy JSON to clipboard for use in Minecraft

- Save animations for later use
